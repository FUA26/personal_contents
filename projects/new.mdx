# Setup Docker Nextjs

Buat sebuah docker compose dan dockerfile untuk run nextjs didalam container docker. Pada tahap ini kita akan membuat untuk kebutuhan development dimana akan menggunakan hot reload. docker-compose.dev.yml

```yaml
version: "3"

services:
  langkah-app:
    build:
      context: .
      dockerfile: dev.Dockerfile
    container_name: langkah-app
    restart: always
    ports:
      - 3000:3000
    volumes:
      - .:/app
      - /app/node_modules
    env_file:
      - .env
```

dev.Dockerfile.yml

```yaml
FROM node:18-alpine

WORKDIR /app

COPY package*.json ./ RUN npm install

COPY . .

RUN npm run build

CMD ["npm", "run", "dev"]
```

Jalankan printah untuk run docker-compose

```bash
# Commend untuk build dan run terpisah
docker-compose -f docker-compose.dev.yml build
docker-compose -f docker-compose.dev.yml up -d

atau
# Commend untuk build sekaligus run
docker-compose -f docker-compose.dev.yml up -d
```
